# BackendPractice
Practice for backend learnings

Basics - practice of basics of NodeJS and ExpressJS (such as server setup, handling HTTP requests such as GET, POST, DELETE)

Exchange Server - Implementation of basics, with Controllers and Routes, to make a server which can provide the following functionalities
    1. Get all currencies
    2. Convert from one currency to another

Usage of Nodemon to enable auto server restart based on detection of updates. 

Fitness Server - Practice of basics, Controllers, Routes. Support the functionalities
    1. support home, abouts and fitness page
    2. get users
    3. get users by gender (using query)
    4. get users by firstName (using params)
